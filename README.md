## OSS CAD Suite Installation:

* [All Intallation Options](https://github.com/YosysHQ/oss-cad-suite-build?tab=readme-ov-file#installation)
* Linux/Ubuntu-specific Installation Example:
```
cd <extracted_location>
wget https://github.com/YosysHQ/oss-cad-suite-build/releases/download/2025-11-10/oss-cad-suite-linux-x64-20251110.tgz
tar -xvf oss-cad-suite-linux-x64-20251110.tgz
export PATH="<extracted_location>/oss-cad-suite/bin:$PATH"
```
